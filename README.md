# Electromagnetic Simulator
--------------
### Website: https://sir-jerm.github.io/electromagnetic/
--------------

- N-Body 3d Electromagnetic Simulator with Boris Algorithm (Leapfrog and Euler Methods included also)
- O(n log n) Time Complexity achieved through Barnes-Hut Algorithm
- Visual Vector Fields, Visual Paths of Charges, 2d Custom Data Graphs
- 10k Particles
- Columb’s Law and Biot-Savart’s Law
- WASD camera movement and rotation (Click and move cursor)
- Home-made 3d renderer
